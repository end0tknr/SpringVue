販売偏差値

販売実績や家族世帯数、販売数から、「販売しやすさ」を算出したものです。
・顧客 多
現在の家族世帯数に対し、販売済の戸数が多い程、○
・競合会社 少
販売済戸数に対し、販売中の不動産会社が少ない程、○ ( ≒ ブルーオーシャン)
・競合物件 少
販売済戸数に対し、販売中の物件数が少ない程、○ ( ≒ ブルーオーシャン)
・商談化 早
物件情報公開～商談開始の日数が短い程、○
・売約数/年
「国交省公開の不動産取引実績」✕「分譲率」
